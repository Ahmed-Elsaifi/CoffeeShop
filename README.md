# Coffee Shop Pro (Vite + React)

Features:
- Vite + React + TailwindCSS
- React Router (Home, Products, Offers, About, Testimonials, Contact, Cart)
- Framer Motion + AOS animations
- Dark/Light mode toggle (icon)
- Cart with localStorage persistence
- 6 sample products and placeholder images

Run locally:
1. unzip the project
2. npm install
3. npm run dev
